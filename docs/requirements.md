# Project Requirements

## Functional Requirements

- Users should be able to create accounts (see [authentication.md](./authentication.md))
- Users should be able to log in and log out securely
- Users should be able to log workouts
- Users should be able to track nutrition entries and goals
- Users should be able to view progress on a dashboard (see [wireframes.md](./wireframes.md))

## Non-Functional Requirements

- Application should be responsive (mobile and desktop)
- User data should remain secure (hashed passwords, scoped queries)
- Navigation should remain simple and user friendly (five main sections)
